How To Use

  1. Select the current game you are playing
  2. Select the game you want the converted sensitivity for
  3. Enter in your sensitivity from your current game
  4. Enter in your current DPI that you use for the current game
  5. Enter in the DPI you will use for the new game
Take the converted sensitivity and continue practicing those flicks and one taps!
